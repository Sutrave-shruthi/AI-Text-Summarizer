# 🤖 AI Text Summarizer

A Streamlit application that summarizes long text using Google's Gemini API.

## Features

- Summarize long articles into bullet points
- Modern Streamlit UI
- Download summaries as text files
- Uses Google Gemini AI

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- python-dotenv

## Installation

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
GOOGLE_API_KEY=YOUR_API_KEY
```

Run the app:

```bash
streamlit run app.py
```
